Given the following property in the <newPropertyContext> tags, relevant information about other properties within the <otherPropertyContext> tags, and the user question provided in the <userQuestion> tags, answer the user question.

When asked about if a property advances housing goals, the percentage of the properties low income units out of the total number of units should be found. If the property has >30% of units marked as low income, it will advance housing goals.

When asked if we have seen the property before, the given property's address should be found. Then, a property with a matching address should be found from the other properties within the <otherPropertyContext> tags. If one of the other properties has a matching address, the new property has been seen before.

When asked if a property qualifies for a Green Loan, the following steps should be taken

1. Find the property has a green building certification and the green building certification agency
2. If the green building certification is yes and the green building certification agency is one of the following, the property qualifies for a Green Loan
Green Building Certification Agencies:
- BREEAM USA
- Build It Green
- Enterprise Community Partners
- Green Building Initiative
- Home Innovation Research Labs
- International Living Future Institute
- Passive House Institute
- Phius (Passive House Institute US)
- U.S. Department of Energy
- International Finance Corporation
- U.S. Environmental Protection Agency
- U.S. Green Building Council
3. Find the properties green building upgrades, if any are in the following list, the property qualifies for a green loan
- Upgrading to ENERGY STARÂ® appliances
- Upgrading boiler
- Replacing inefficient lighting
- Installing solar systems
- Installing water-saving irrigation systems
- Improving insulation
- Making other energy- & water-saving improvements
4. Return if property is a green loan and justification, and mention that this information is extracted from High-Performance Building Report.
